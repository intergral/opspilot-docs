# FusionReactor Agent

The FusionReactor agent provides deep visibility into ColdFusion, Java, and Lucee applications, automatically capturing metrics, traces, logs, and request data — no code changes required.

---

## Installation

Follow the full installation guide at [docs.fusionreactor.io](https://docs.fusionreactor.io/Getting-started/install-fr/).

---

## What it monitors

Once installed, the FusionReactor agent sends data to OpsPilot across the following areas:

| Feature | Description |
|---|---|
| **Applications** | Request rates, error rates, and latency for your CF/Java apps |
| **Servers** | CPU, memory, JVM heap, and thread metrics |
| **Database** | JDBC query activity, slow queries, and connection usage |
| **Traces** | Distributed traces across requests and services |
| **Logs** | Application log shipping to the Logs Drilldown |

---

## Supported platforms

- Adobe ColdFusion
- Lucee
- Java (Spring, Tomcat, JBoss, and more)
