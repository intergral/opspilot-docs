# Incidents

Incidents linked to this service.

The **Incidents** tab shows the [incidents](../Incidents/overview.md) associated with a service, so you can see what is currently or recently affecting it without leaving the service view.

Use the **Catalog** dropdown to select a catalog, then choose a service to view its incidents.
