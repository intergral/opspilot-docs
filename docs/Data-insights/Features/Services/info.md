# Info

Catalog info: ownership, tier, dependencies, and blast radius.

The **Info** tab surfaces the [Catalog](../../../Admin-and-data/Catalog/catalog.md) metadata for a service, so you can see who owns it, how critical it is, and what depends on it, all without leaving the service view.

Use the **Catalog** dropdown to select a catalog, then choose a service to view its catalog info.
