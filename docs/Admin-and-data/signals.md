# Signals

Control which OpenTelemetry signal types are ingested into your OpsPilot account. Navigate to **Data and Licenses > Signals** to open it.

Disabling a signal type will block all incoming data of that type from being collected or stored.

| Signal | Description |
|---|---|
| **Logs** | Application and infrastructure log data |
| **Metrics** | Numeric time-series data from your services and infrastructure |
| **Traces** | Distributed trace data for request flow analysis |
| **AI** | AI-powered analysis and Coworker features |

Toggle any signal off to stop ingesting that data type. Changes may take up to 15 minutes to take effect.
