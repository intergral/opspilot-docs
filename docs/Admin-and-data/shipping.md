# Shipping

Configure where OpsPilot sends your telemetry data. Navigate to **Data and Licenses > Shipping** to open it.

## Simple mode

| Field | Description |
|---|---|
| **Endpoint URL** | The URL of your OpenTelemetry collector (such as, `https://otel-collector.example.com`) |
| **Protocol** | The transport protocol - gRPC or HTTP |
| **Headers (optional)** | Any additional headers required by your collector (such as, `Authorization=Bearer xxx`) |

Click **Save** to apply the configuration.

## Advanced mode

Switch to the **Advanced** tab for additional configuration options.
